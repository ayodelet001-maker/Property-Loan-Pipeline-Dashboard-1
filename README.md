# Property-Loan-Pipeline-Dashboard-1
17000+ UK development loans; cleaned & visualized in Excel. Data was cleaned by removing duplicates, standardizing texts using text to Columns &amp; Proper(), filled missing values and added 2 new columns 1: Period (Year) = Loan Term/12.  2: Interest Revenue = Application Value X Rates(%) X Period (Year). A subsequent analysis &amp; Dashboad was created 
Analysis Dashbord: 1: Success Rate by Region 2: YoY Revenue & Application. 3: Average SQFT of Funded Projects. 
Interactive Excel Dashboard (Using slicers, Pivot Charts, and conditional formatting) evaluates: 1. Best Pipeline: (Top Regions/Brokers by conversion) 2. Most Successful sorces (Revenue & Volume share)
KPIs: Total interest revenue (£63,047,086) Total Successful Project (402) Average SQFT funded (8736)
Key Insights: NorthWest leads with 60.15% success. Interest Revenue peaked in 2022 with £22,434,687 Total interest, making it the most successful year recorded. Direct source proved the most successful channel with 61% success.
